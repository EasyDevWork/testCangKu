🚀 快速开始指南

步骤 1: 下载项目

git clone https://github.com/EasyDevWork/testCangKu.git
cd testCangKu

步骤 2: 构建 APK

cd AndroidDemo
chmod +x gradlew
./gradlew assembleDebug

步骤 3: 安装到设备

adb install app/build/outputs/apk/debug/app-debug.apk

或者使用构建脚本

chmod +x build-apk.sh
./build-apk.sh
